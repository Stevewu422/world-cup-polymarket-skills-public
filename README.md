# World Cup Polymarket Skills (Sanitized Public Package)

This repository contains sanitized public Hermes skills for World Cup match research, responsible betting workflows, Polymarket execution runbooks, and pool accounting.

## Included skills

- `world-cup-polymarket-orchestrator` — end-to-end World Cup + Polymarket workflow orchestration.
- `world-cup-daily-prediction` — daily/twice-daily World Cup prediction and monitoring routines.
- `world-cup-caiqiu-prediction` — World Cup football/caiqiu prediction workflow.
- `polymarket-pool-accounting` — participant pool accounting and market-level allocation.
- `polymarket-pool-ledger-accounting` — ledger-style pool accounting.
- `sports-betting-risk-management` — responsible sports betting risk framework.

## Security / compliance

This is a **sanitized public copy**. It intentionally redacts:

- private keys / API tokens / cookies / passphrases
- exact private operational endpoints
- transaction hashes and order hashes
- local credential files and runtime artifacts

Use it as a workflow template. Do not treat it as financial advice. Always verify legal availability, venue restrictions, wallet safety, orderbook liquidity, and bankroll limits before any live trade.
