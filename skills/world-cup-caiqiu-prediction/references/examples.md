# 世界杯彩球 Skill 使用示例

## 手动即时预测

用户：

```text
今天世界杯彩球怎么下
```

助手执行：
1. 核验未来 24-36 小时赛程。
2. 复盘过去 24 小时比赛。
3. 查询赔率/盘口/Polymarket。
4. 输出逐场执行卡。

## Polymarket 输出示例

```text
Polymarket：Brazil vs Germany / Brazil Yes / 当前 0.54 / 链接 https://polymarket.com/event/...
执行条件：Brazil Yes ≤0.50 才考虑；高于 0.58 放弃。
```

如果找不到：

```text
Polymarket：未找到该场直接市场；不伪造链接。
```
